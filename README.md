\# Telegram Gifts Marketplace (TON Mini App)



Static prototype for GitHub Pages.



\## Deploy

1\. Create repo `telegram-gifts-market`

2\. Upload all files (keep folder structure)

3\. Enable GitHub Pages (main branch /root)

4\. Open https://your-username.github.io/telegram-gifts-market

5\. Open in Telegram → Attach Mini App



\## Features

\- Real TON Connect (Tonkeeper, MyTonWallet, etc.)

\- Live balance from toncenter.com

\- Buy → real TON transfer to demo address

\- Sell / My Gifts / Activity (localStorage)

\- Filters, smooth UI, Telegram theme



\*\*DEMO NOTE\*\*: Transactions send real TON to a demo address. Use tiny amounts or test wallets only.



Replace manifest URL after deploy!
